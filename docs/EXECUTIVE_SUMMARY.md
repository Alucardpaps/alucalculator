# Executive Summary: Alucalculator 2.0 Transformation

## 1. Strategic Vision
To transition Alucalculator.com from a generic "weight calculator" to the industry-standard **"Digital Machine Engineer's Handbook"**.

## 2. Engineering & Production Reality
* **Dynamic Density:** Moving away from static 2.70 g/cm³ to alloy-specific gravity (ASTM B209).
* **Cost Reality:** Incorporating Price-per-kg logic to output financial data, not just mass.
* **Nesting:** Helping engineers understand how many parts fit in a standard stock length (6m/12ft).

## 3. Technical Robustness
* **Offline-First:** PWA architecture allowing use in signal-dead zones (factory floors).
* **Validation:** Strict input sanitization to prevent calculation errors and security risks.

## 4. Competitive Moat

| Current State | Target State |
|---------------|--------------|
| Single density assumption | Alloy-aware calculations |
| Weight-only output | Weight + Cost + Nesting |
| Online-dependent | Offline-capable (PWA) |
| No validation | Strict input sanitization |
